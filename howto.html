Howto Git

A small start:

  1. List branches: git branch

  2. Add branch: git branch script-changes #s-c or any other name

    . change branch: git checkout script-changes #or the name listed in git branch

Change something in a file, then:

  1. git add a-file

  2. commit -m "type some info" #commits all your added files

  3. git push --set-upstream origin script-changes #send to git/cloud

  4. Change branch: git checkout master

  5. Update main: git pull #fetch and merge from git/cloud
